# 📊 Interactive Sales Dashboard in Excel

## 📌 Project Overview
This project is an interactive Excel dashboard designed to analyze sales performance using Pivot Tables, Pivot Charts, Slicers, and KPIs. It helps users monitor business performance and make data-driven decisions.

## ✨ Features
- Interactive Dashboard
- Pivot Tables
- Pivot Charts
- Slicers
- KPIs
- Dynamic Filtering
- Monthly & Yearly Sales Analysis

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## 📈 Dashboard Insights
- Total Sales
- Total Profit
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Top Performing Products
